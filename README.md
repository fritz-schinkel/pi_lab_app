# pi_da_tut
Digital Annealer Tutorial on Raspberry PI Zero 2W

Install your Rapsberry Pi 2W with "lite" image e.g. download Raspberry Pi Imager on your PC from https://www.raspberrypi.com/software/ and start the software. Select the model ``Raspberry Pi Zero 2W``. Under ``Raspberry PI OS (other)`` you will find ``Raspberry PI OS Lite (64-bit)`` or ``Raspberry PI OS (Legacy, 64-bit) Lite``. Both are suited, the second one ist older and more stable. Select one of those and transfere the image to a micro sd card. Define a suitable. Define the requested content for hostname, user and access to your local WLAN.
